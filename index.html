Cómo activar modo escritorio en Brave
1
Abre Brave en tu móvil y ve a brave://flags
2
Busca Desktop mode → activa Default to desktop mode on Android
3
O bien: menú ⋮ → marca Sitio de escritorio mientras estás en open.spotify.com
4
Para que sea permanente solo para Spotify: Configuración → Sitio → open.spotify.com → Modo escritorio: ON
5
Instala esta página como app: menú ⋮ → Añadir a pantalla de inicio → se abre directo como app sin barra de navegación
